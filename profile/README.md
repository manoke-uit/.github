### 📌 About the Project

**Manoke** is a karaoke app with AI-based scoring and social features like:
- 🔐 Auth & friend system  
- 🤖 AI singing score evaluator  
- 🔔 Notifications & score sharing  
- 📄 Backend powered by NestJS & PostgreSQL
